CSV File Requirements

For the import to succeed, your CSV file must match the following format guidelines:

Required Columns
  • Asset Type: Laptop, Desktop, Printer, Monitor, Scanner, Router, Switch, Server, UPS, Other
  • Brand: Brand manufacturer (e.g. Dell, HP, Apple)
  • Model: Model name (e.g. Latitude 5420, MacBook Pro)
  • Serial Number: Must be unique across all inventory records
  • Purchase Date: Formats supported: YYYY-MM-DD or MM/DD/YYYY
  • Warranty Expiry: Formats supported: YYYY-MM-DD or MM/DD/YYYY
Optional Columns
  • Status: Available, Repair, Scrap, Lost (Defaults to Available if blank)
  • Location: Physical location or room (e.g. HQ - Room 402, Remote)
  • Remarks: General details or technical observations
Sample Formatting Example
Type,Brand,Model,Serial,Purchase Date,Warranty Expiry,Status,Location
Laptop,Dell,Latitude,SN-DELL-102,2025-01-15,2028-01-15,Available,HQ - Room 402
Monitor,HP,E24i,SN-HP-3049,2024-06-20,2027-06-20,Available,Remote
